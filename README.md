# 🎲 3D Rolling Cube 404 Page Animation

A creative **404 Not Found** page built using **HTML** and **CSS**,
featuring a smoothly animated **3D rolling cube**. The project
demonstrates how modern CSS transforms, perspective, and keyframe
animations can create engaging UI experiences without JavaScript.

## ✨ Preview

> A stylish 3D cube continuously rolls while displaying a custom 404
> error page, making broken links feel visually engaging instead of
> frustrating.

## 🚀 Features

-   🎲 Smooth 3D rolling cube animation
-   🎨 Pure HTML & CSS implementation
-   📱 Responsive layout
-   ⚡ Lightweight and fast
-   🧩 Easy to customize colors, size, speed, and text
-   🌐 Works in all modern browsers

## 🛠️ Tech Stack

-   HTML5
-   CSS3
    -   CSS 3D Transforms
    -   Perspective
    -   Keyframe Animations
    -   Flexbox
